1. restart.sh is the running script of Linux system
2. start.bat and stop.bat are running scripts for Windows system
3. Before the Windows system starts, modify the corresponding java jdk1.8 installation path
     start.bat -> Modify the path of JAVA_EXE=D:\"Program Files"\Java\jre8\bin\java
4. Before starting the Linux system, please configure the JAVA_HOME environment variable
